# bshivaramakrishnan.github.io

Personal portfolio website.

## Structure

```
bshivaramakrishnan.github.io/
├── index.html                        ← Main portfolio
└── projects/
    ├── backup-system.html            ← Enterprise Backup System
    ├── codebase-rag.html             ← CodeBase RAG Engine
    ├── ai-tcp-transfer.html          ← AI-Optimized TCP Transfer
    ├── jarvis-cli.html               ← Jarvis CLI
    ├── distributed-cache.html        ← Distributed Cache System
    └── neural-classifier.html        ← Neural Networks Image Classifier
```

## Setup

1. Your 6 existing project HTML files go into `projects/` with the exact filenames above
2. Push to GitHub → live at `https://bshivaramakrishnan.github.io`

## Deploy

```bash
git init
git add .
git commit -m "feat: deploy portfolio"
git remote add origin https://github.com/bshivaramakrishnan/bshivaramakrishnan.github.io.git
git branch -M main
git push -u origin main
```

Then go to **Settings → Pages → Source: main branch → Save**.
